# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/microsoft/purview-dlm-mcp/issues) to track bugs and feature requests. Please search the existing issues before filing new issues to avoid duplicates. For new issues, file your bug or feature request as a new issue.

For help and questions about using this project, please use GitHub Issues.

For security vulnerabilities, please see [SECURITY.md](SECURITY.md).

## Microsoft Support Policy

Support for this project is limited to the resources listed above.
